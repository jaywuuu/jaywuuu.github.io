# jaywu's TypeScript Blog

A modern, lightweight blog engine that renders Markdown files dynamically using TypeScript.

## Features

- **Markdown-Driven**: Write posts in standard Markdown (`.md`) and they are automatically rendered.
- **TypeScript Powered**: The frontend is built with TypeScript for type safety and better maintainability.
- **Git Integration**: Blog post dates are automatically extracted from the Git commit history.
- **Single Page Application (SPA)**: Smooth routing between the post list and individual posts using URL hashes (e.g., `#/post/hello-world`).
- **Responsive Design**: A clean, typography-focused layout built with Bootstrap 4 and custom CSS.

## Project Structure

- `posts/`: Store your Markdown blog posts here.
- `src/index.ts`: The main TypeScript application logic.
- `js/`: Contains the compiled JavaScript files (generated by `tsc`).
- `scripts/generate-posts.js`: A Node.js script that scans the `posts/` directory and generates `posts.json`.
- `posts.json`: Metadata and content for all blog posts (generated automatically).
- `index.html`: The main entry point and shell for the application.
- `css/main.css`: Custom styles for the blog.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (for the build process)
- [Git](https://git-scm.com/) (to manage posts and track commit dates)

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Adding New Posts

1. Create a new `.md` file in the `posts/` directory (e.g., `my-new-post.md`).
2. Start the file with a Level 1 Header (`# My Title`) which will be used as the post title.
3. Commit the file to Git to ensure a "last modified" date is captured:
   ```bash
   git add posts/my-new-post.md
   git commit -m "Add new blog post"
   ```

### Building the Site

To update the blog with new posts and recompile the TypeScript code, run:

```bash
npm run build
```

This command runs two steps:
1. `npm run generate`: Runs the script to rebuild `posts.json` from the `posts/` directory.
2. `npx tsc`: Compiles the TypeScript source code into browser-ready JavaScript in the `js/` folder.

## Technical Details

- **Markdown Rendering**: Uses the [marked](https://marked.js.org/) library.
- **Routing**: Handled via `window.location.hash` and the `hashchange` event listener in `src/index.ts`.
- **Date Extraction**: The generation script uses `git log -1 --format=%ai` to fetch the precise date of the last commit for each file.
